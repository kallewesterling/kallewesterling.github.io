Skillbuilding website

Django-based full-stack website

As a project manager for the Digital Humanities Research Institute, I built two websites from the ground up using the static site-builder Jekyll. The first one was the project‚Äôs main website, providing all interested with basic information about the project and its goals. The website is still live at dhinstitutes.org.

The other website was built as a micro-site that provided access to all the information presented at the Association for Computing in the Humanities conference in 2019, and is still live at ach.dhinstitutes.org.

The most challenging task of building a website followed the COVID-19 pandemic in 2020. My robust bespoke virtual learning environment for our skills modules, administration, and skills assessment was complemented by custom-written JavaScript, CSS styling using Bootstrap 5.
